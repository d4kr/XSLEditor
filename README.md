# XSLEditor

A local desktop tool for developers to edit multi-file XSLT/XSL-FO templates, generate PDFs on demand, and perform advanced debugging.

## Pipeline

```
XML → XSLT → XSL-FO → PDF
```

## Purpose

Reduce the edit-to-preview cycle (target < 5s) with no dependency on external systems. Everything runs locally.
