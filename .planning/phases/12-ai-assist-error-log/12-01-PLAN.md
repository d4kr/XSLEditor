<objective>
Add a 5th column to the error log TableView (after Message) containing a small button per row.
Clicking the button assembles an Italian prompt from the row's LogEntry.message(), URL-encodes it,
and opens https://chatgpt.com/?q=... in the default browser via XSLEditorApp.hostServices().showDocument().
The button must not trigger row selection, editor navigation, or any mutation of the entry lists.

Purpose: Developers can ask ChatGPT about any pipeline error with a single click, without leaving
the tool or manually copy-pasting the message.

| T-12-01 | Tampering | LogController — URL construction | mitigate | URLEncoder.encode(prompt, StandardCharsets.UTF_8) followed by .replace("+", "%20") ensures all special characters (spaces, ampersands, hash, etc.) are percent-encoded before the URL is passed to showDocument(); the browser never receives raw user content as URL structure |

| T-12-03 | Denial of Service | Very long error messages | accept | ChatGPT URL query length is handled by the browser and ChatGPT's own limits. The tool is local-developer-only; no user input from untrusted sources reaches this path |

Insert this block (per D-01, D-02, D-03, D-04, D-05):
```java
        // 5. AI assist column cell factory (ERR-06 / D-01..D-05)
        colAi.setCellFactory(col -> new TableCell<LogEntry, Void>() {
            private final Button btn = createAiButton();

            @Override
            protected void updateItem(Void item, boolean empty) {
                super.updateItem(item, empty);
                if (empty || getTableRow() == null || getTableRow().getItem() == null) {
                    setGraphic(null);
                } else {
                    setGraphic(btn);
                }
            }

            private Button createAiButton() {
                Button b = new Button("💬"); // 💬 Unicode chat bubble
                b.setTooltip(new Tooltip("Ask ChatGPT about this error"));
                b.setStyle("-fx-padding: 1 4 1 4; -fx-font-size: 11;");
                b.setFocusTraversable(false);
                b.setOnAction(evt -> {
                    // D-05: consume event so the row click handler is not triggered
                    evt.consume();
                    LogEntry entry = getTableRow().getItem();
                    if (entry == null) return;
                    // D-02: Italian preamble + raw message
                    String prompt = "Ho questo errore nel mio progetto XSLT/XSL-FO, puoi aiutarmi?\n\n"
                            + entry.message();
                    // D-03: URL-encode, replace "+" with "%20" for compatibility
                    String encoded = URLEncoder.encode(prompt, StandardCharsets.UTF_8)
                            .replace("+", "%20");
                    String url = "https://chatgpt.com/?q=" + encoded;
                    XSLEditorApp.hostServices().showDocument(url);
                });
                return b;
            }
        });
```
