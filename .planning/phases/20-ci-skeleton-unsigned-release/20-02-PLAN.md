## Failure handling

| Symptom | Likely cause | Fix |
|---------|-------------|-----|
| `build-jar` fails on Gradle | `./gradlew` needs execute permission | Add `- run: chmod +x gradlew` before the Gradle step |
| `package-windows` fails "WiX tools not found" | `echo "${WIX}bin" >> $GITHUB_PATH` not reached | Check step order; verify `$WIX` is set on runner |
| `package-windows` fails version format | Pre-release suffix not stripped | Verify `sed 's/-.*//'` step runs before jpackage |
| `release` job fails "Resource not accessible" | Missing `permissions: contents: write` | Check the permissions block is on the `release` job |
| Assets not attached | `release-assets/**/*` glob not matching | Try `files: |` multiline list with explicit paths per artifact dir |
