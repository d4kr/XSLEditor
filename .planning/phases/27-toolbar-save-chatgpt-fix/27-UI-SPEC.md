## Interaction Contracts

### Save Button Disable Binding

- Binding target: `saveButton.disableProperty()`
- Bind expression: `activeEditorTab.dirty.not()` when a tab is open; `true` (hardcoded disable) when no tab is open
- Rebind trigger: same tab-switch listener already used for `undoButton`/`redoButton` — extend the existing rebind method to include `saveButton`
- Tab close: `saveButton.setDisable(true)` via the null-tab path in the rebind method

### ChatGPT Button Click Flow

1. User clicks 💬 button on any error log row
2. `addEventHandler(MOUSE_PRESSED, ...)` fires: ButtonBase arms internally, then handler runs
3. Browser opens with `https://chatgpt.com/?q=<encoded-error-message>`
4. Event does NOT bubble to TableView row selector (handler calls `mouseEvt.consume()`)

### Tab-Switch Rebind Method

- Existing `rebindUndoRedo()` (or renamed `rebindToolbarButtons()` at Claude's discretion) is extended to also rebind `saveButton`
- On tab selection change: unbind previous, bind to new active tab's `dirty` property
- On null selection (all tabs closed): set all toolbar buttons to `disable=true`
