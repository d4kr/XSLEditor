### New component (this phase only)

| Component | JavaFX type | Specification | Source |
|-----------|-------------|---------------|--------|
| App icon ImageView | `ImageView` | fitWidth=64, fitHeight=64, preserveRatio=true; loaded via `getClass().getResourceAsStream("/ch/ti/gagi/xsleditor/icon.png")`; inserted as **first element** in root VBox, before `titleLabel`; centered via `VBox.setAlignment(Pos.CENTER)` or wrapping `HBox` (discretion per CONTEXT.md) | CONTEXT.md D-04 |

Icon load contract:
- If `getClass().getResourceAsStream(...)` returns null or `Image.isError()` is true: omit the ImageView silently — dialog must not throw or show a broken-image placeholder.
- If valid: add to VBox as position 0 (before titleLabel).

Stage icon contract (XSLEditorApp):
- Load `javafx.scene.image.Image` from `/ch/ti/gagi/xsleditor/icon.png` via `getClass().getResourceAsStream(...)`.
- Check `image.isError()` before use.
- Call `primaryStage.getIcons().add(image)` if valid.
- If load fails: log a warning (exact mechanism at Claude's discretion per CONTEXT.md — Java Logger, System.err, or silent no-op are all acceptable).
- Must execute before `primaryStage.show()` (currently line 49 of `XSLEditorApp.java`).
