| Boundary | Description |
|----------|-------------|
| About dialog → external browser | Hyperlink hands a hardcoded URL to `hostServices.showDocument(...)`, which delegates to the OS default browser. URL is a constant string baked at compile time, no user input. |

## STRIDE Threat Register

This phase changes only documentation/metadata and a single hardcoded URL string. No new code paths, no input parsing, no new attack surface.

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-28-01 | Tampering | LICENSE file | accept | LICENSE is a public legal text checked into git; tampering is detected via git history and PR review. No runtime exposure. |
| T-28-02 | Information Disclosure | AboutDialog hyperlink → opensource.org/licenses/MIT | accept | URL is a hardcoded string constant, not user-controlled, points to a well-known public site. No PII transmitted. |
| T-28-03 | Spoofing | README logo `<img src="src/main/resources/...">` | accept | Image path is a relative repo path resolved by GitHub's renderer; no remote fetch, no executable content. |

Threat surface for this phase is effectively zero; ASVS L1 has no applicable controls beyond the existing repo-wide ones.
</threat_model>
