```xml
          accelerator="F5"

```

Attribute order convention: fx:id, text, accelerator, onAction.

Cross-platform token: use `Shortcut+` (NOT `Ctrl+` or `Meta+`).
JavaFX resolves `Shortcut` → Cmd on macOS, Ctrl on Windows/Linux via KeyCombination.valueOf().

KBD-04 / KBD-05 special case: `menuItemSetEntrypoint` and `menuItemSetXmlInput` have NO onAction
in FXML. Their action handlers are bound by FileTreeController.initialize() via setOnAction().
Add ONLY the accelerator attribute — do NOT add onAction.
