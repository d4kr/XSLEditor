## Spacing Scale

JavaFX uses integer pixel values. All values are multiples of 4.

| Token | Value | JavaFX API | Usage in this dialog |
|-------|-------|------------|----------------------|
| xs | 4 | `new Insets(4)` / `setSpacing(4)` | Column gap between label and version in GridPane |
| sm | 8 | `new Insets(8)` / `setSpacing(8)` | VBox spacing between logical sections |
| md | 16 | `new Insets(16)` | DialogPane content padding (top/bottom) |
| lg | 24 | `new Insets(24, 16, 16, 16)` | DialogPane overall padding (Insets top,right,bottom,left) |

**DialogPane padding:** `new Insets(24, 16, 16, 16)` — 24 top, 16 sides and bottom.

**VBox spacing:** 8 between all direct children of the root VBox.

**GridPane column spacing:** hgap=16 (between label column and version column), vgap=4 (between rows).

**Separator margins:** `VBox.setMargin(separator, new Insets(4, 0, 4, 0))` — 4px top and bottom, no horizontal margin.

Exceptions: none.

## Typography

JavaFX font values use `Font.font(family, weight, size)` or `-fx-font-*` inline style.

| Role | Size | Weight | JavaFX declaration | Usage |
|------|------|--------|--------------------|-------|
| App title | 18px | Bold | `Font.font("System", FontWeight.BOLD, 18)` | "XSLEditor  v0.1.0" label |
| Section header | 12px | Bold | `Font.font("System", FontWeight.BOLD, 12)` | "Runtime Stack" label |
| Body / data | 13px | Normal (400) | `Font.font("System", 13)` | Version values in GridPane, author line, license line |
| Label column | 13px | Normal (400) | `Font.font("Monospaced", 13)` | Library names (Java, Saxon-HE, FOP, JavaFX) — monospaced for column alignment |

**Font family rationale:**
- "System" resolves to the platform UI font (San Francisco on macOS, Segoe UI on Windows) — consistent with the rest of the app.
- "Monospaced" for the GridPane label column provides natural tabular alignment without a custom cell factory.

**Hyperlink:** Inherits body size (13px); rendered with default JavaFX hyperlink styling (underline, accent color).

## Color

Derived from `main.css` — all values match the established VS Code dark theme palette already in use.

| Role | Value | JavaFX property | Usage |
|------|-------|-----------------|-------|
| Dominant (60%) — dialog background | `#2b2b2b` | `-fx-background-color` on DialogPane | Dialog surface |
| Secondary (30%) — text primary | `#cccccc` | `-fx-text-fill` | Body text, version values, author line |
| Muted text | `#888888` | `-fx-text-fill` | Section header "Runtime Stack", GridPane label column names |
| Accent (10%) — hyperlink | `#007acc` | JavaFX default `.hyperlink` color (VS Code blue already present via log-filter-btn:selected) | License URL hyperlink only |
| Separator | `#444444` | `-fx-background-color` on Separator | Two horizontal rule lines |

**Accent reserved for:** the license Hyperlink node only. No other element in this dialog uses the accent color.

**No destructive color needed** — this dialog has no destructive actions.

**How to apply background:** Set `-fx-background-color: #2b2b2b;` on the DialogPane via `getDialogPane().setStyle(...)`. The dialog inherits the scene but the DialogPane chrome must be explicitly styled to match.

## Copywriting Contract

All strings are exact — executor must use these verbatim.

| Element | Exact Copy | Notes |
|---------|-----------|-------|
| Dialog window title | `About XSLEditor` | `setTitle("About XSLEditor")` |
| App title line | `XSLEditor  v0.1.0` | Two spaces between name and version; version read from `version.properties` at runtime (D-01) |
| Section header | `Runtime Stack` | Plain Label, no colon |
| GridPane row 1 — label | `Java` | Left column |
| GridPane row 1 — value | Runtime value from `System.getProperty("java.version")` | Right column |
| GridPane row 2 — label | `Saxon-HE` | Left column |
| GridPane row 2 — value | Runtime value from `net.sf.saxon.Version.getProductVersion()` | Right column |
| GridPane row 3 — label | `FOP` | Left column |
| GridPane row 3 — value | Runtime value from `org.apache.fop.Version.getVersion()` | Right column; fallback: `"2.9"` if API unavailable |
| GridPane row 4 — label | `JavaFX` | Left column |
| GridPane row 4 — value | Runtime value from `System.getProperty("javafx.version")` | Right column |
| Author line | `Author: d4kr` | Single Label node — source: git config user.name |
| License label | `License: Apache 2.0` | Inline with Hyperlink — rendered as Label + Hyperlink in an HBox |
| License hyperlink text | `View license` | Hyperlink node text |
| License URL | `https://www.apache.org/licenses/LICENSE-2.0` | Opened via `getHostServices().showDocument(url)` (D-05) |
| Close button | `Close` | `ButtonType.CLOSE` — JavaFX default label |
| Header text | (none) | `setHeaderText(null)` — consistent with SearchDialog |

**Empty state:** Not applicable — dialog always renders static content; no data-dependent empty state exists.

**Error state:** If `version.properties` cannot be read, display `"unknown"` for the version value without throwing an exception. No error UI is shown to the user for this case.

**Destructive actions:** None — no confirmation flow required.

## Interaction Contract

| Interaction | Behavior |
|-------------|----------|
| Open | `AboutDialog.showAndWait()` called from `MainController.handleAbout()` |
| Modality | `initModality(Modality.APPLICATION_MODAL)` — blocks main window (D-04) |
| Owner | `initOwner(primaryStage)` — dialog centered over main window (D-04) |
| Close button | `ButtonType.CLOSE` — dismisses dialog, returns `Optional.empty()` |
| Escape key | JavaFX default — closes dialog (same as Close button) |
| License hyperlink click | `hostServices.showDocument("https://www.apache.org/licenses/LICENSE-2.0")` opens default browser (D-05); dialog stays open |
| Hyperlink cursor | Default JavaFX hyperlink cursor (hand pointer) — no override needed |
| Resize | `setResizable(false)` — fixed-size informational dialog |
| Dialog preferred width | 360px — `getDialogPane().setPrefWidth(360)` |
